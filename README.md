# DarkBotX

## GETTING STARTED
Git clone the project then run the command below

```
./setup.sh
```

## Run the project
```
uv run main.py
```


## Troubleshoot
Incase the setup fails, you can manually install uv on your system then run the script below to get started.

```
uv init --python 3.12
uv add torch genesis-world
```
